# 主线大纲

系统流 的主线应从一个能立刻展示题材承诺的事件开始,再通过三到四个阶段扩大压力。每个阶段都要改变主角的选择空间,而不是只增加信息量。大纲写作时要把外部事件、人物欲望和长期风险绑在一起。

## 节奏规则

- 开篇三章内必须让读者确认题材承诺和主角困境
- 每个小阶段至少有一次关系、资源、身份或认知的不可逆变化
- 信息释放要分层,避免一次性解释完世界和冲突
- 单章结尾保留情绪钩子或信息钩子,但不能靠虚假悬念

## 主线阶段

1. **进入压力场**: 主角被卷入 系统流 的核心问题,旧生活秩序被打破。
2. **建立行动方法**: 主角找到暂时可用的解决方式,同时暴露该方式的局限。
3. **代价扩大**: 对手、环境或关系网络开始反制,主角必须放弃某种安全选择。
4. **重排秩序**: 主角用新的判断完成阶段性胜利,但留下下一卷更大的结构问题。

## 待规划章节

- 第 1 章: 用具体事件展示题材承诺和主角当前缺口
- 第 2 章: 让主角第一次主动尝试解决,并暴露规则边界
- 第 3 章: 引入关键配角或敌对力量,扩大冲突范围
- 第 4 章: 让前面选择产生后果,推动主线进入下一阶段
