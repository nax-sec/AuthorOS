# 风格规则

memory-curator 按 delta 追加,不重写。

## 已确立风格规则

- 克制冷静,细节为先,主角内心独白限制在每章 200 字以内
- 视角统一(第一人称侦探视角或第三人称受限视角),避免全知漂移
- 线索描写必须客观可读,关键线索给读者公平看到的机会
- 章尾留疑(疑/险/反转/启示 四选一)
- 解释式独白尽量化为对话或推理过程

## 已禁止写法

- 作者全知抢镜(突然解释只有作者知道的事)
- 读者看不到的"私藏线索"用来反转
- 主角神算式破案(无错路、无弱点)
- 大段心理独白代替推理
- AI 味、流水账、空泛悬疑感堆砌
