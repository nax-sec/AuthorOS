# 风格规则 (style)

memory-curator 按 delta 追加,不重写。

## 已确立风格规则

- 史诗调性 + 现代节奏融合,庄重而不冗长
- 多线视角清晰切换,避免视角漂移
- 魔法/神迹描写有触感(温度、痛感、声响),不停留在抽象
- 章尾留钩(疑/险/启示/抉择 四选一)
- 适度引用"古老誓约""灰烬记忆"等史诗修辞,不堆砌

## 已禁止写法

- 中世纪伪欧式翻译腔(全篇 thou/thee 化中文)
- 大段世界设定干叙
- 种族塑造止于标签(矮人=爱酒、精灵=高傲)
- AI 味、流水账、空泛史诗感堆砌
